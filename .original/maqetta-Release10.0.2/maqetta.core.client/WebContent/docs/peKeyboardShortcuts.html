Page Editor Keyboard Shortcuts

The following shortcut keyboard accelerators are available in the Maqetta page editor:

Ctrl(Win)/Cmd(Mac)ccopy - copy the currently selected widget to your clipboard.
Ctrl(Win)/Cmd(Mac)sSave - save the current file.
Ctrl(Win)/Cmd(Mac) + ShiftsSave As - save the current file under a new name.
Ctrl(Win)/Cmd(Mac)vpaste - paste the widget from your clipboard (click paste, then click your cursor where you want to make the paste).
Ctrl(Win)/Cmd(Mac)xcut - delete the currently selected widget and copy it to your clipboard.
Ctrl(Win)/Cmd(Mac)zUndo - undo the last change. Can undo multiple changes.
Ctrl(Win)/Cmd(Mac) + ShiftzRedo - cancel the last "Undo". Can redo multiple "Undo"s.
BS
(on Win, DEL key also)		delete - delete the currently selected widget.
Ctrl(Win)/Cmd(Mac)0Preview in Browser - displays the current page in a new browser window, as it will be rendered in the broswer (as opposed to how it looks in the Page Editor).
Ctrl(Win)/Cmd(Mac)1Display Design - display the file in design view (default view).
Ctrl(Win)/Cmd(Mac)2Display Source - display the file in source view. See Source Editing for more on working in source view.
Ctrl(Win)/Cmd(Mac)3Split Vertically - split the canvas window vertically to show both design and source views.
Ctrl(Win)/Cmd(Mac)4Split Horizontally - split the canvas horizontally to show both design and source views.
 F1Help - opens Maqetta's documentation set in a new browser tab/window
