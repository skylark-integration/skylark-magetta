Widget Palette: Mobile Data Store Based (Basic)

EdgeToEdgeDataList: Comma separated data

Steps

  1. Create a new HTML file with composition type iPhone
  2. Open the Dojox Mobile folder in the Widget palette.
  3. Drag a View widget onto the canvas.
  4. Drag an EdgeToEdgeDataList widget onto the canvas.
  5. SmartInput is displayed. Choose Comma separated data from the drop down list.
  6. Change the data in the text area to: 
    7. Label 1, transitionToPage
Label 2, transitionToPage
Label 3, transitionToPage
Label 4, transitionToPage
  7. Click Finish.
  8. The EdgeToEdgeDataList should be updated with the data you entered.
  9. Save and close the file.
  10. Reopen the file
  11. Click on the preview icon in the toolbar, this should open a new browser window and the grid should be displayed with the data entered

Expected results

  1. The EdgeToEdgeDataList in page editor and preview should be displaying the data you just entered.

EdgeToEdgeDataList: Data file from workspace

Steps

  1. Open the Dojox Mobile folder in the Widget palette.
  2. Drag a View widget onto the canvas.
  3. Drag an EdgeToEdgeDataList widget onto the canvas.
  4. SmartInput is displayed. Choose Data file from workspace from the drop down list on the first panel of the wizard.
  5. Click on the folder icon.
  6. The select a source dialog will be displayed, navigate to the samples/sample_data folder and then select the mobile_edge_to_edge_or_round_rec__data_list.json file. Click the OK button
  7. The EdgeToEdgeDataList should be updated with the data from the file.
  8. Save and close the file.
  9. Reopen the file
  10. Click on the preview icon in the toolbar, this should open a new browser window and the EdgeToEdgeDataList should be displayed with the data from the file.

Expected results

  1. The EdgeToEdgeDataList in page editor and preview should be displaying the data from the file.
