Widget Palette Dojo GridX Tests (Basic)

Comma separated data

Steps

  1. Choose New HTML File from the New menu
  2. Open the Dojo Controls folder in the Widget palette.
  3. Drag the GridX widget onto the canvas.
  4. SmartInput is displayed. Choose Comma separated data from the drop down list on the first panel of the wizard.
  5. Change the data in the text area to: 
    6. one, two, three
data 1, data 2, data 3
xxxx, yyy, zzz
111, 222, 3333
  6. Click Finish.
  7. The GridX should be updated with the data you entered. In addition, the grid's width should be 100%.
  8. Save and close the file.
  9. Reopen the file
  10. Click on the preview icon in the toolbar, this should open a new browser window and the grid should be displayed with the data entered

Expected results

  1. The GridX in page editor and preview should be displaying the data you just entered.
  2. You should see 3 columns with the headers one, two and three.
  3. You should see 3 rows of data, corresponding to the data you entered.

Smart Input (GridX Configuration Wizard)

Steps

  1. Choose New HTML File from the New menu
  2. Open the Dojo Controls folder in the Widget palette.
  3. Drag the GridX widget onto the canvas.
  4. SmartInput is displayed.
  5. Verify the SmartInput is a modal dialog containing a wizard.
  6. Verify the wizard contains three panels (one for data source set-up, one for column selection, and one for previewing the grid and resizing/reordering columns):
    • Click the Next button 2 times and then the Back button two times to verify you can go to the end of the wizard and back (seeing the 3 steps along the way).
    • Along the top of the dialog there should be a clickable heading for each step. Click on each heading and verify the corresponding wizard page is selected.
  7. On the first panel:
    • Mouse down and hold on the icon in the button right corner of the text area. Drag to re-size the the text area.
    • Click on the ? icon to display the help.
    • Click on the ? icon to hide the help.

Expected results

  1. The user should see a modal dialog containing a wizard.
  2. The user should be able to navigate through the 3 steps of the wizard with Next/Back buttons or the clickable headings.
  3. On the first panel, the user should be able to re-size the text area and display/hide help.

Canceling the wizard

Steps

  1. Choose New HTML File from the New menu
  2. Open the Dojo Controls folder in the Widget palette.
  3. Drag the GridX widget onto the canvas.
  4. SmartInput is displayed. Choose Comma separated data from the drop down list
  5. Change the data in the text area to:
    6. one, two, three
data 1, <b>data 2</b>, data 3
xxxx, yyy, zzz
111, 222, 3333
  6. NOTE: The UI should not change despite the presence of HTML tags. Unlike DataGrid, GridX does not support specifying the use of Plan Text vs. HTML Markup.
  7. Click the Cancel button.
  8. The GridX should be be displayed with the default data, not the data you entered.
  9. Save and close the file.
  10. Reopen the file
  11. Click on the preview icon in the toolbar, this should open a new browser window and the grid should be displayed with the default data.

Expected results

  1. The GridX in page editor and preview should be displaying the default data.

Data file from workspace

Steps

  1. Choose New HTML File from the New menu
  2. Open the Dojo Controls folder in the Widget palette.
  3. Drag the GridX widget onto the canvas.
  4. SmartInput is displayed. Choose Data file from workspace from the drop down list on the first panel of the wizard.
  5. Click on the folder icon.
  6. The select a source dialog will be displayed, navigate to the samples/sample_data folder and then select the htmlSample.json file. Click the OK button
  7. The text area should now display the text samples/sample_data/htmlSample.json Unlike DataGrid, radio buttons to specify plain text or HTML markup should not be displayed.
  8. Click the Finish button on the wizard.
  9. The GridX should be updated with the data from the htmlSample file. Notice that the data is displayed with HTML markup rendered (e.g., Fruits is made bold).
  10. Save and close the file.
  11. Reopen the file
  12. Click on the preview icon in the toolbar, this should open a new browser window and the grid should be displayed with the data from the file.

Expected results

  1. The GridX in page editor and preview should be displaying the data from the file.

Data from URL (JSONP)

Steps

  1. Continuing from the test case above, open the HTML File from the Data from workspace test case.
  2. Double click on the GridX Widget in the design view to display the smart input.
  3. SmartInput is displayed. Choose Data from URL (JSONP) from the drop down list on the first panel.
  4. In the text area enter the following URL: http://maqettaservice.appspot.com/sampleservice
  5. Click the Finish button on the wizard.
  6. The GridX should be updated with the data from the service. Notice that the data is displayed with HTML tags rendered (for example Fruits).
  7. Save and close the file.
  8. Reopen the file
  9. Click on the preview icon in the toolbar, this should open a new browser window and the grid should be displayed with the data from the service.

Expected results

  1. The GridX in page editor and preview should be displaying the data from the file.

Comma separated data (with empty cells)

Steps

  1. Choose New HTML File from the New menu
  2. Open the Dojo Controls folder in the Widget palette.
  3. Drag the GridX widget onto the canvas.
  4. SmartInput is displayed. Choose Comma separated data from the drop down list on the first panel of the wizard.
  5. Change the data in the text area to the following (note empty cells in the 2nd column of the first two rows): 
    6. Date de début,Date de fin,Code UT,Intitulé UT
03/03/2012,,UT3,Unité de Travail Trois
02/02/2011,,UT2,Unité de Travail Deux
01/01/2010,31/12/2012,UT1,Unité de Travail Un
  6. Click Finish.
  7. The GridX should be updated with the data you entered to show 4 columns of data.
  8. Double-click the GridX to reopen smart input. Ensure that the data looks the same as when you pasted it in originally. In particular, ensure the 2nd column is present and the empty cells in the first two rows are reflected.
  9. Save and close the file.
  10. Reopen the file
  11. Click on the preview icon in the toolbar, this should open a new browser window and the grid should be displayed with the data entered

Expected results

  1. The GridX in page editor and preview should be displaying the data you just enter.
  2. You should see 4 columns with the headers specified in your data.
  3. You should see 3 rows of data, corresponding to the data you entered.
  4. Reopening smart input should show exactly the same data as what you pasted in originally.
