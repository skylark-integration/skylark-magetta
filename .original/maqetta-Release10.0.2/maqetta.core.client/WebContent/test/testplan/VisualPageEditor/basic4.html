Visual page editor basic tests (part 4 - Drawing Tools)

Shape creation - simple drag/drop from widget palette

Steps

  1. Standard setup things: Clear the browser cache. Launch the Maqetta server. Login if necessary. In some cases, you'll want to setup a virgin user environment by creating a whole new user login.
  2. Go to preferences and look at "Snap to nearest widget" setting.
    It should be on by default.
  3. Click on the "New file" icon on the main toolbar. Set composition type to "Desktop hi-fi sketch". Choose any filename.
  4. Drag/drop each of the Drawing Tools widgets from the Widgets palette onto the canvas
    • Text should show SmartInput to edit the text context
    • Line and Arrow should create a horizontal line
    • Polyline and Polyline Arrow should create a default 2-segment polyline
    • Rectangle, Rounded Rectangle and Circle should create a default shape sized around 100px
    • You should see snap lines during dragging
  5. Click on each of the Drawing Tools widgets in the Widgets palette, followed by clicking on the canvas. See bullets below for follow-on clicking for polyline widgets.
    • Text, Line, Arrow, Rectangles and Circle: same results as previous step
    • The polylines should act differently. The first click should drop the first point of polyline. Click on canvas 3 more times. You should now have a 3-segment polyline. Click once more on top of the last point of polyline. This should complete the polyline and exit "polyline creation" mode.
    • You should see snap lines as the mouse moves around canvas before first click (i.e., after clicking on a widget in widget palette but before first click on canvas to add widget)
  6. Click on each of the Drawing Tools widgets in the Widgets palette, followed by a drag operation on the canvas. See bullets below for follow-on clicking for polyline widgets.
    • Text isn't resizable, so should result in same operation as previous steps.
    • Line, Arrow, Rectangles and Circle: the new widget should be the size of the drag area
    • The polylines should act differently. The first drag should define the size of the first segment. After the drag, click on canvas 2 more times. You should now have a 3-segment polyline. Press the ESC key. This should complete the polyline and exit "polyline creation" mode.
    • You should see snap lines as the mouse moves around canvas before first click (i.e., after clicking on a widget in widget palette but before first click on canvas to add widget)
  7. Repeat the above test with an extra wrinkle. Click on each of the Drawing Tools widgets in the Widgets palette, followed by a drag operation on the canvas. While dragging, hold down the shift key to constrain the drag. See bullets below for follow-on clicking for polyline widgets.
    • Text isn't resizable, so should result in same operation as previous steps.
    • Line, Arrow, Polyline, Polyline Arrow: dragging should force the line segment to be either horizontal or verticle
    • Rectangles and Circle: dragging should force the shape into a square aspect ratio (i.e., w=h).

Shapes as containers

Steps

  1. Click on the "New file" icon on the main toolbar. Set composition type to "Desktop hi-fi sketch". Choose any filename.
  2. Drag/drop one of each of a Rectangle, Rounded Rectangle and Circle onto canvas.
  3. Drag/drop a Button into each of the above three widgets.
  4. For each of the Rectangle, Rounded Rectangle and Circle, select that widget and move it.
    • The Buttons inside of the shapes should move with the shape.
    • In the Outline palette, you should see that the Buttons are child widgets of each of the shapes.
