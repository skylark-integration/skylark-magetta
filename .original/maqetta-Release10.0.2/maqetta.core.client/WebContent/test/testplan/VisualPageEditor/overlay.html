Visual page editor basic tests ("overlay" Dijit widget support)

Tooltip placement

Steps

  1. Standard setup things: Clear the browser cache. Launch the Maqetta server. Login if necessary. In some cases, you'll want to setup a virgin user environment by creating a whole new user login.
  2. Click on the "New file" icon on the main toolbar. Choose any filename.
  3. Drag a Button onto the canvas. Set the text on the Button to "Show Dialog"
  4. Drag a Tooltip on to the Button. The Toolbar will be associated with the Button. Put text in the tooltip like "click here to show the dialog"

Expected results

  1. Click on the Tooltip in the Outline view and it should be made visible, adjacent to the Button.
  2. Click on another element and the Tooltip should go away.
  3. Examine the States view. In addition to the "Normal" state, you should see a state representing the Tooltip visible state with a unique ID, e.g. _show:dijit_Tooltip_96.

Dialog placement

Steps

  1. Continuing from the previous example, drag a Dialog on the canvas.
  2. Drag a Button widget onto the Dialog. Set the label to "Close".

Expected results

  1. Click on another element and the Dialog should go away.
  2. Click on the Dialog in the Outline view and it should be made visible again.
  3. Examine the States view. In addition to the "Normal" and Tooltip states, you should see a state representing the Dialog visible state with a unique ID, e.g. _show:dijit_Dialog_0.

Dialog states

Steps

  1. Select the "Show Dialog" Button and go to the events section in the properties palette. Set the onclick event to set the _show state for the Dialog.
  2. Select the "Close" Button in the Dialog from the outline view and go to the events section in the properties palette. Set the onclick event to set the Normal state.

Expected results

  1. Go to preview mode.
  2. Hover over the "Show Dialog" button. You should see the Tooltip pop up.
  3. Click on the "Show Dialog" Button. The dialog should appear.
  4. Click on the "Close" Button. The dialog should disappear.
