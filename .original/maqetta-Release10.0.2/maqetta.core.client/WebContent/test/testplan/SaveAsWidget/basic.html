Save-as Widget basic tests

Create a new Custom Widget

Steps

  1. Standard setup things: Clear the browser cache. Launch the Maqetta server. Login if necessary. In some cases, you'll want to setup a virgin user environment by creating a whole new user login.
  2. Click on Create---> "Desktop Application" on the toolbar. Choose any filename.
  3. Add a DIV element to the page
  4. Add some dojo widgets to the DIV element. Use 3-4 different widgets.
  5. Select the DIV by click on it.
  6. Click the dropdown next to Save. Select "Save As Widget"
  7. Type in a group and a name for the widget and click OK.

Expected results

  1. A "User Widgets" section will appear in the Widget palette with the new widget you just created.

Add a custom widget to a HTML page

Steps

  1. Create a new HTML/desktop application
  2. Locate the custom widget you created in the previous section and add it to the page
  3. Save the page. Close and reopen.

Expected results

  1. The custom widget is added to the page. Click on each sub widget and make sure they aren't selected. Only the entire custom widget should be selected.
